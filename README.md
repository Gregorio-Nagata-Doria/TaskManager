<h1>Projeto simples de gerenciamento de tarefas desenvolvido em C#</h1>

  O programa utiliza um arquivo json para armazenar as tarefas e edita ele conforme os comandos inseridos pelo usuário.

  Comandos:
    <li>
       <ul>
        add: cria uma tarefa nova após o usuário inserir seu texto
      </ul>
       <ul>
        update: atualiza o texto da tarefa desejada
      </ul>
       <ul>
        delete: deleta a tarefa escolhida
      </ul>
       <ul>
        mark_in_progress, mark-done: muda o status de uma tarefa para o status escolhido
      </ul>
       <ul>
         list: lista todas as tarefas
      </ul>
       <ul>
        list done,list todo, list in-progress: lista as tarefas com o status escolhido
      </ul>
      
    </li>
