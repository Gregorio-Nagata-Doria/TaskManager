<h1>Projeto simples de gerenciamento de tarefas desenvolvido em C#</h1>

  O programa utiliza um arquivo json para armazenar as tarefas e edita ele conforme os comandos inseridos pelo usuário.

  Comandos:
  <h4>        add: cria uma tarefa nova após o usuário inserir seu texto</h4>
<h4>        update: atualiza o texto da tarefa desejada</h4>
<h4>        delete: deleta a tarefa escolhida</h4>
<h4>
        mark_in_progress, mark-done: muda o status de uma tarefa para o status escolhido</h4>
<h4>         list: lista todas as tarefas</h4>
<h4>        list done,list todo, list in-progress: lista as tarefas com o status escolhido</h4>
